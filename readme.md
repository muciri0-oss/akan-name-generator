# Akan Name Generator

## Description
This is a simple web application that calculates the day of the week a user was born and displays their Akan name based on their gender. Akan names originate from Ghanaian culture, where children are named according to the day of the week they are born.

## Author
Peterson Muciri

## Setup Instructions
1. Clone or download this repository
2. Open the project folder
3. Open `index.html` in a web browser

## BDD (Behavior Driven Development)
- User enters their birth date
- User selects their gender
- User clicks the submit button
- Application calculates the day of the week
- Application displays the correct Akan name
- Application alerts the user if inputs are invalid
